## üöÄ How to execute the project

To clone and run this app, you will need [Git](https://git-scm.com) and [NodeJs](https://nodejs.org/en/) installed on your computer.

### üåÄ Cloning the repository

```bash
# Clone the repository
$ git clone https://github.com/cristianoprogramador/rockethelp.git

# Access the project folder in terminal/cmd
$ cd moveit
```

### üé≤ Running the application

```bash
# Install the depedencies
$ yarn install

# Run the Application in Development
$ yarn dev

# The server will start on port:3333 - go to http://localhost:3333
```
